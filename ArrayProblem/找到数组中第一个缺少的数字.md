### 找到数组中第一个缺少的数字
> 给定一个数组，长度，元素都是从0到n。但是中间缺少了一个值，找到这个值
> [0,1,2,3,4,6,7,8,9] 返回5

**思路**

先加出来当前数组的总和
再计算出本来应该的数组的总和
相减得到一个差值
这个差值表示了缺少值和多出来这个值的差值
多出的这个值一定是size
所以用size减去这个差值就是缺少的值。

**代码**

```python
def get_miss_num(nums: list):
    count = 0
    for i in range(0, len(nums)):
        count += nums[i]
        count -= i
    return len(nums)-count
```
