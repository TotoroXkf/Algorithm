# 在行列都排好序的矩阵中找数
> 如题

**思路**
--------------------

从右上角开始找
因为行和列有序，向下表示增大，向左表示减少
所以遇到数字，判断大小，然后决定走向就可以

**代码**
--------------------

```python
def search_in_matrix(matrix:list, target:int):
    if matrix and matrix[0]:
        row = 0
        col = len(matrix[0]) - 1
        while row < len(matrix) and col > -1:
            value = matrix[row][col]
            if value == target:
                return True
            elif value < target:
                row = row + 1
            else:
                col = col - 1
    return False
```
